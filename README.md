# Enhanced Armenian ALPR System
**Student:** Eduard Arakelyan  
**Course:** M1 Artificial Intelligence - UFAR  

## 🚀 Project Description
A Neuro-Symbolic approach to Automatic License Plate Recognition (ALPR). We combine **Deep Learning** (YOLOv11) for spatial perception with **Knowledge Representation** (OWL Ontology via `owlready2`) to ensure 100% logical accuracy for Armenian plate formats according to Decision No. 1251-N.

## 📁 Structure
- `src/main.py`: Core Python implementation.
- `requirements.txt`: Library dependencies.

## 🛠️ How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the main script: `python src/main.py`
